## Installation
1. Clone this repository and go to QAD folder
```bash
git clone https://github.com/KNU-LMAP/edge-qad.git
cd edge-qad
```
2. Create a conda environment and install requirements
```bash
conda create -n qad python=3.8 -y
conda activate qad
pip install -upgrade pip
pip install -e .
```
